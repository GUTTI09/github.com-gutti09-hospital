# 🏥 Hospital Management System (Python)

A **console-based Hospital Management System** developed using **Python**, designed to manage patient registration, appointments, payments, emergency services, and ambulance management with persistent data storage.

---

## 📌 Project Overview

This project simulates real-world hospital operations by digitizing:
- Patient handling
- Doctor appointments
- Emergency and ambulance services
- Online consultation
- Digital prescriptions
- Payment and revenue management

All data is stored permanently using **JSON file handling**, ensuring data is not lost after program restarts.

---

## ✨ Features

### 👤 Patient Management
- Patient registration with:
  - Email validation
  - 10-digit mobile number validation
  - Duplicate mobile number check
  - Aadhaar-style validation
  - OTP verification
- Unique Patient ID generation
- Secure patient login

### 🩺 Appointment Management
- Book appointments with doctors from multiple departments
- Appointment types:
  - Normal
  - Emergency
  - Online Consultation
- View, reschedule, and cancel appointments

### 🚨 Emergency & Ambulance Services
- Emergency service activation
- Automatic ambulance booking
- Live ambulance tracking simulation
- Ambulance fee billing

### 💬 Online Consultation & 📄 Digital Prescription
- Online consultation for remote diagnosis
- Digital prescription generation after consultation

### 💳 Payment & Billing
- Department-wise consultation fees
- Date & time stamped payments
- Ambulance billing
- Permanent payment history storage

### 📊 Admin Features
- View all registered patients
- View appointment history
- Hospital revenue calculation

---
